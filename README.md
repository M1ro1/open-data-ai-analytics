Тема: Аналіз ринку транспортних засобів в Україні.



Мета: Створення інструментарію для обробки та аналізу даних МВС щодо реєстрації транспортних засобів.



Джерело даних: https://data.gov.ua/dataset/06779371-308f-42d7-895e-5a39833375f0/resource/3f13166f-090b-499e-8e23-e9851c5a5f67



Питання та гіпотези:



Популярність електрокарів: Чи спостерігається зростання кількості реєстрацій електромобілів протягом останнього звітного періоду порівняно з аналогічним періодом минулого року?

Вікова структура автопарку: Яка частка нових автомобілів серед усіх первинних реєстрацій.

Колірні вподобання та бренди: Які три марки автомобілів є найпопулярнішими в Україні, та чи існує кореляція між маркою авто та його кольором?

Статус: Дослідження завершено.
